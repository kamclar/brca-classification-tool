# BRCA Classification Tool

Podpůrný nástroj pro klasifikaci BRCA1/BRCA2 variant podle ENIGMA VCEP ACMG/AMP guidelines.

## 🎯 Cíl projektu

Nástroj pro podporu klasifikace germline variant v genech BRCA1 a BRCA2. Není plně automatický klasifikátor — finální rozhodnutí zůstává na genetikovi/lékaři.

## 📁 Struktura

```
├── docs/               ← Dokumentace (GitBook)
├── notebooks/          ← Jupyter/Colab notebooky
├── src/                ← Zdrojový kód
└── references/         ← Externí materiály
```
